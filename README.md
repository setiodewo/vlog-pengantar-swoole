# vlog-pengantar-swoole
File2 yg digunakan di vlog Orang-IT tentang Swoole

1. Jangan lupa chmod file run.sh supaya bisa dijalankan.
   ```
   chmod +x run.sh
   ```

2. Jalankan dengan cara:
   ```
   ./run.sh
   ```
